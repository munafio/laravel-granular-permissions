# Laravel Granular Permissions Backend

## Getting Started

-   Clone the repository.
-   Install the dependencies

```
composer install
```

-   Update `.env` database credentials with yours.
-   Run the migrations with the seeding

```
php artisan migrate --seed
```

-   Run the server

```
php artisan server
```
